## 2. 專案規格表 (Specifications)

| 類別 | 規格內容 | 說明 |
| :--- | :--- | :--- |
| **框架** | React 19 / Vite 6 | 高效開發環境與最新渲染特性。 |
| **語言** | TypeScript 5.7 | 強型別檢查，減少運行時錯誤。 |
| **通訊** | Supabase SDK / Realtime | 提供 OAuth、資料庫查詢及即時訊息推送。 |
| **樣式** | Tailwind CSS 4 + Radix UI | 自動化樣式管理與無障礙 UI 元件。 |
| **導航** | React Router 7 | 動態路由管理與分頁加載。 |
| **圖標** | Lucide React | 標準化向量圖標庫。 |
| **狀態** | Zustand / TanStack Query | 全域狀態與伺服器端快取同步。 |
